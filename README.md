# AllInOne-RN 
## An Ecommerce Mobile Application

This is an ecommerce application built with React Native. It allows users to browse products, add them to a cart, and checkout using a secure payment gateway.

## Features

- User authentication and authorization
- Product browsing and searching
- Product sorting and filtering
- Shopping cart management
- Secure checkout process
- Order history and status tracking

## Installation

To run this application locally, you'll need to have React Native installed on your machine.

1. Clone this repository:
   ```git clone https://github.com/your-username/ecommerce-app.git```
   
2. Install dependencies:
   ```npm install```
   
3. Start the development server to run on Android:
   ```npx react-native run-android```
   
4. Start the development server to run on IOS:
   ```npx react-native run-ios```
   

## Usage

To use this allinone-ecommerce application, open application on emulator or real device through ADB. You'll be presented with the home page, where you can browse products and add them to your cart. To checkout, click the "Checkout" button and follow the prompts to enter your shipping and payment information.
